## 💡 프로그램 소개

**네이버 블로그 HTML 복사기**는 ChatGPT, Gemini 등 AI가 생성한 HTML 코드를 네이버 블로그 스마트에디터 ONE에 바로 붙여넣을 수 있도록 **서식 있는 텍스트(Rich Text)** 로 변환하여 클립보드에 복사해주는 크롬 확장 프로그램입니다.

기존에는 AI가 만들어준 HTML을 네이버 블로그에 적용하려면 별도 HTML 편집기를 사용하거나 복잡한 과정을 거쳐야 했습니다. 이 확장 프로그램을 사용하면 **클릭 한 번**으로 변환부터 클립보드 복사까지 자동 완료되어, 네이버 블로그 편집기에서 `Ctrl + V`만 누르면 됩니다.

특히 AI 출력물에 자주 포함되는 **찌꺼기(각주, 코드블록 기호, 파일명 등)를 자동으로 제거**하고, `[제목]`, `[인용]`, `[체크]` 등 **커스텀 태그를 네이버 블로그 호환 HTML로 자동 매핑**하는 고급 변환 엔진을 내장하고 있습니다.

## 🚀 주요 기능

- **HTML → Rich Text 변환**: 날것의 HTML 코드를 텍스트가 아닌 **서식(Rich Text)** 형태로 클립보드에 복사하여, 네이버 블로그 에디터에서 서식이 유지된 채 붙여넣기 가능
- **AI 찌꺼기 자동 정리**: Gemini·ChatGPT 등의 출력에 포함된 `[cite:N]`, ` ```html `, 이미지 파일명, PDF 태그 등을 자동 제거
- **커스텀 태그 자동 변환**: `[제목]`, `[인용]`, `[구분선]`, `[체크]`, `[사진]` 등의 직관적인 괄호 태그를 네이버 블로그에 최적화된 HTML로 자동 매핑
- **마크다운 호환**: `## 제목`, `> 인용`, `- 리스트`, `**볼드**` 등 마크다운 문법도 함께 변환 지원
- **클릭 한 번 완료**: 팝업 UI에서 HTML을 붙여넣고 버튼 하나만 누르면 변환 → 클립보드 복사까지 자동 완료
- **네이버 블로그 완벽 호환**: 스마트에디터 ONE의 블록 분리 이슈 등을 고려한 세밀한 HTML 출력

## 🏗️ 시스템 아키텍처

확장 프로그램의 HTML 변환 처리 흐름입니다.

```text
┌─────────────────────────────────────────────────────────────────────┐
│  Chrome Extension (popup.js)                                        │
│                                                                     │
│  ┌─ [1단계] AI 찌꺼기 청소 ───────────────────────────────────────┐ │
│  │  • [cite:N] 각주 제거                                          │ │
│  │  • ```html / ```markdown 코드블록 기호 제거                    │ │
│  │  • Gemini 이미지 파일명 / PDF 태그 제거                        │ │
│  │  • 태그 줄바꿈 보정 (앞뒤 강제 개행)                           │ │
│  │  • 링크·해시태그 줄바꿈 보정                                   │ │
│  ├─────────────────────────────────────────────────────────────────┤ │
│  │                                                                 │ │
│  ┌─ [2단계] 태그 & 마크다운 → HTML 변환 엔진 ────────────────────┐ │
│  │  • [제목] / ## → <h2>                                          │ │
│  │  • [인용] / > → <blockquote>                                   │ │
│  │  • [구분선] / --- → <hr>                                       │ │
│  │  • [체크] / ✅ / - → ✅ 리스트                                 │ │
│  │  • [사진] → 📷 이미지 삽입 위치 표시                           │ │
│  │  • **볼드** → <strong>                                         │ │
│  │  • 일반 텍스트 → <p> 래핑                                      │ │
│  ├─────────────────────────────────────────────────────────────────┤ │
│  │                                                                 │ │
│  ┌─ [3단계] 클립보드 복사 (Rich Text) ───────────────────────────┐ │
│  │  • Blob(text/html) 생성                                        │ │
│  │  • ClipboardItem API로 서식 유지 복사                          │ │
│  │  • 네이버 에디터에서 Ctrl+V로 즉시 붙여넣기 가능               │ │
│  └─────────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## 🛠 기술 스택

| 구분 | 기술 | 설명 |
| :--- | :--- | :--- |
| **플랫폼** | Chrome Extension (Manifest V3) | 최신 크롬 확장 프로그램 표준 |
| **프론트엔드** | HTML5 + Vanilla CSS | 경량 팝업 UI |
| **로직** | JavaScript (ES6+) | 정규식 기반 변환 엔진 |
| **클립보드** | Clipboard API (`navigator.clipboard.write`) | Rich Text 형태로 서식 유지 복사 |
| **권한** | `clipboardWrite` | 클립보드 쓰기 권한만 사용 |

## ⚙️ 설치 방법

크롬 웹스토어에 정식 등록 전이므로, **개발자 모드**를 통해 간편하게 설치할 수 있습니다.

| 단계 | 설명 |
| :---: | :--- |
| **1** | 본 저장소 우측 상단의 초록색 **[Code]** → `Download ZIP` 클릭 후 압축 해제 |
| **2** | 크롬 브라우저에서 `chrome://extensions/` 접속 |
| **3** | 우측 상단 **[개발자 모드]** 토글 ON |
| **4** | **[압축해제된 확장 프로그램을 로드합니다.]** 클릭 |
| **5** | 압축 해제한 `NaverBlogHTMLCopier` 폴더 선택 |
| **6** | 크롬 우측 상단 퍼즐 아이콘 → 확장 프로그램 **고정(📌)** |

## 📝 사용 예시

```text
1.  크롬 우측 상단의 확장 프로그램 아이콘 클릭
2.  텍스트 박스에 AI가 만들어준 HTML 코드를 붙여넣기
3.  [네이버 블로그용 복사하기] 버튼 클릭
4.  네이버 블로그 포스팅 쓰기 화면에서 Ctrl + V
```

**입력 예시:**
```html
[제목] 여름 다이어트 식단 가이드
[인용] 건강한 다이어트는 굶는 것이 아닌, 올바르게 먹는 것입니다.
[구분선]
[체크] 아침: 그릭 요거트 + 견과류
[체크] 점심: 닭가슴살 샐러드
[체크] 저녁: 연어 스테이크 + 채소
[사진]
**꾸준함이 가장 중요합니다!**
```

**변환 결과:** 네이버 블로그에 서식이 적용된 깔끔한 포스트로 즉시 붙여넣기 됩니다.

## 🔄 지원하는 태그 및 변환 규칙

| 입력 (괄호 태그 / 마크다운) | 변환 결과 | 설명 |
| :--- | :--- | :--- |
| `[추천제목] 텍스트` | 💡 초록색 강조 블록 | 추천 제목 하이라이트 |
| `[제목] 텍스트` / `## 텍스트` / `■ 텍스트` | `<h2>` 제목 | 네이버 제목 블록 |
| `[인용] 텍스트` / `> 텍스트` | `<blockquote>` | 인용구 블록 (연속 인용 자동 병합) |
| `[구분선]` / `---` | `<hr>` | 구분선 |
| `[체크] 텍스트` / `✅ 텍스트` / `- 텍스트` | ✅ 리스트 항목 | 체크 리스트 |
| `[썸네일 사진]` / `[사진]` | 📷 이미지 삽입 위치 | 이미지 자리표시자 |
| `**텍스트**` | `<strong>` | 볼드 텍스트 |
