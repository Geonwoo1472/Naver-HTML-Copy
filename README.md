# 네이버 블로그 HTML 복사기 (Naver HTML Copy Extension)

네이버 블로그 스마트에디터 ONE에 HTML 코드를 손쉽게 서식 있는 텍스트(Rich Text) 형태로 붙여넣을 수 있게 도와주는 크롬 확장 프로그램(Chrome Extension)입니다. 
AI가 만들어준 HTML 코드나 복잡한 표, 서식이 있는 텍스트를 네이버 블로그에 바로 적용하고 싶을 때 유용합니다.

## ✨ 주요 기능
- **HTML을 서식 있는 텍스트로 변환**: 날것의 HTML 코드를 클립보드에 복사할 때, 텍스트 형태가 아닌 **서식(Rich Text)** 형태로 복사합니다.
- **클릭 한 번으로 처리**: 입력창에 HTML을 넣고 버튼을 누르면 즉시 변환되어 클립보드에 저장됩니다.
- **네이버 블로그 호환**: 네이버 블로그 스마트에디터 ONE 편집기에 `Ctrl + V`로 바로 붙여넣을 수 있습니다.

## 🛠 설치 및 사용 방법 (개발자 모드)

크롬 웹스토어에 정식 등록되기 전이므로, **개발자 모드**를 통해 손쉽게 설치할 수 있습니다.

1. 본 저장소(Repository)의 우측 상단 초록색 **[Code]** 버튼을 눌러 `Download ZIP`으로 파일을 다운로드하고 압축을 풉니다.
2. 크롬 브라우저에서 `chrome://extensions/` 에 접속합니다.
3. 우측 상단의 **[개발자 모드]** 토글을 켭니다.
4. 좌측 상단의 **[압축해제된 확장 프로그램을 로드합니다.]** 버튼을 클릭합니다.
5. 다운로드하여 압축을 푼 폴더(`NaverBlogHTMLCopier`)를 선택합니다.
6. 크롬 브라우저 우측 상단의 퍼즐 모양 아이콘을 눌러 "네이버 블로그 HTML 복사기"를 고정(핀 꽂기)하여 사용합니다.

## 📝 사용 예시
1. 확장 프로그램 아이콘을 클릭합니다.
2. 텍스트 박스에 AI(ChatGPT 등)가 짜준 HTML 코드를 복사해서 붙여넣습니다. (예: `<h1>제목</h1><p>내용입니다.</p>`)
3. **[네이버 블로그용 복사하기]** 버튼을 클릭합니다.
4. 네이버 블로그 포스팅 쓰기 화면에서 `Ctrl + V`로 붙여넣습니다.
